About the creation

The website is a personal project created by Estela Valle with the aims to learn and practice programming skills. It is open source and it is available on Estela's GitHub profile.

Website future improvements
Currently I am working on the website login mechanism, as well as improving the chords formatter, and increasing test coverage for existing functionalities.
Technologies used in the website

Currently, the website counts with a PostgreSQL database, which is accessible through the search engine created using Flask and SQLAlchemy.

Main technologies used in the project:

- HTML, Javascript and Twitter bootstrap, through flask templates

- Unit tests using pytest running in docker

- Site is open source, available on github at https://github.com/Ex-tela/Magical_Chords_Project

- ChordPro format parser and html chord generator written in python

- PostgreSQL as database
